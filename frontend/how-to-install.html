Install WebShield Extension

Follow these steps to add WebShield to your browser for real-time protection.

1

Open Google Chrome and go to chrome://extensions/

2

Enable Developer mode (toggle in the top right corner)

3

Click Load unpacked

4

Select the folder: extension from your computer

5

The extension will now appear in your browser's extension list!
